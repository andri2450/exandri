Diagnostic Checker - DNS, Email & Website

Cek kendala DNS, indikasi kirim/terima email, dan akses website pada domain kamu. Berjalan 100% di browser (statis, cocok untuk GitHub Pages).

Cek ini berbasis DNS publik dan tes koneksi HTTP/HTTPS dari browser. Cek port SMTP/IMAP/POP3 secara langsung tidak bisa dilakukan dari browser karena batasan keamanan browser — untuk itu perlu versi server-side (PHP/Node).

Hasil bersifat indikatif berdasarkan DNS publik dan tes dari browser, bukan pengganti pengecekan log server secara langsung.
